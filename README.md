# Echoes of the Falling Shards

Echoes of the Falling Shards is a dark-fantasy oriental Idle RPG.
Built with React, Vite, Tailwind CSS, and Capacitor for Android deployment.

## Caratteristiche
- Auto-Battler e meccaniche RPG incrementali.
- Sistema avanzato di forgiatura, rarità ed equipaggiamento.
- "Torre dei Demoni" e spedizioni offline.
- Interfaccia reattiva e grafiche ad alta definizione (VFX).

## Requisiti
- Node.js >= 18
- Capacitor CLI
- Android Studio (per la build APK)

## Installazione e Avvio
1. Installa le dipendenze:
   ```bash
   npm install
   ```
2. Avvia in locale (Browser):
   ```bash
   npm run dev
   ```
3. Compilazione Android:
   ```bash
   npm run build
   npx cap sync android
   ```
   Apri in Android Studio con `npx cap open android`.
